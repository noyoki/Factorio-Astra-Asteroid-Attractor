
# Astra Asteroid Attractor
Astra Asteroid Attractor adds a space-platform-only beacon that attracts more asteroids.  Intended for space-platform focused gameplay.  

Note:  Only works on stationary platforms.  Very power-hungry.

## Features
- Every N ticks, a wave of asteroids will be generated north-ish of the platform proportional to per-tick quantities at random-ish angles. 
- Effects stack additively per asteroid attractor
- Unlocked after beacons + asteroid collectors

## Compatability
- All items/recipes/etc added in this mod are given prefixed names that shouldn't overlap with anything else. 
- This uses asteroid spawn definitions, so it should automatically work with custom space_locations or custom asteroids.
